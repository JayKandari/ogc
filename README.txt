1. ABOUT
===========

Open Graph Comments module allows to show open graph (og) tag of particular
link in comment. It get link from comment body and fetches og meta tags
information for og:title, og:link, og:image and og:description. And appends
this information to original comment body.

2. REQUIREMENTS
================

* Core comment module.

3. INSTALLATION
================

* Install as usual, 
  see https://drupal.org/documentation/install/modules-themes/modules-7 for 
  further information.
